IPv4, Subnetting, and IPv6 Flashcards

Click on any term or question below to reveal the answer.

Total number of bits in an IPv4 address

32 bits, split into 4 octets.

The 8-bit binary weights for IPv4 conversion

128, 64, 32, 16, 8, 4, 2, 1.

Primary function of a subnet mask

To split an IP address into its network and host portions.

First address in any subnet

The network identifier (cannot be assigned to a host).

Last address in any subnet

The broadcast address (cannot be assigned to a host).

Formula for calculating usable hosts in a subnet

Total addresses minus 2 (subtracting the network and broadcast addresses).

Number of usable hosts in a /24 subnet

254 usable hosts.

Number of usable hosts in a /30 subnet

2 usable hosts.

Default mask for a Class A network

/8

Default mask for a Class B network

/16

Default mask for a Class C network

/24

Purpose of the IPv4 Class D range (224.0.0.0/4)

Multicast routing (sending from one to a subscribed group).

IPv4 Loopback address block

127.0.0.0/8 (typically 127.0.0.1).

CIDR (Classless Inter-Domain Routing)

A routing method that drops rigid class boundaries, allowing subnet masks of any length (0-32).

VLSM (Variable Length Subnet Masking)

A technique allowing different subnets within the same network to have different mask sizes, eliminating address waste.

RFC 1918 Class A private range

10.0.0.0/8

RFC 1918 Class B private range

172.16.0.0/12

RFC 1918 Class C private range

192.168.0.0/16

APIPA / link-local IPv4 range

169.254.0.0/16 (assigned when DHCP fails).

Supernetting / Route summarization

The process of merging contiguous smaller network blocks into a single larger block to reduce routing table size.

DHCP DORA process steps

Discover, Offer, Request, Acknowledge.

DHCP T1 Timer

Triggered at 50% of the lease duration; the client attempts to renew the lease via unicast.

PAT (NAT-overload)

A form of translation mapping many private IP addresses to a single public IP address, using unique port numbers to separate sessions.

Total size of an IPv6 address

128 bits, providing 2128 total addresses.

IPv6 Link-local prefix

fe80::/10 (auto-configured per interface).

1. If an octet is 11000000 in binary, what is its decimal value?

  • ◯ A) 192
  • ◯ B) 128
  • ◯ C) 224
  • ◯ D) 168

Correct Answer: A) 192

Rationale: The leftmost two bits correspond to the weights 128 and 64, which sum together to reach this value.

Hint: Think about the specific decimal weights of the first two positions from the left in an 8-bit sequence.

2. How many usable hosts are available in a subnet with a /29 CIDR mask?

  • ◯ A) 6
  • ◯ B) 8
  • ◯ C) 14
  • ◯ D) 2
Show Answer & Rationale

Correct Answer: A) 6

Rationale: A /29 mask provides a total block size of 8, and subtracting the network and broadcast addresses yields the usable count.

Hint: Remember to subtract the network identifier and the broadcast address from the total block size.

3. Which of the following IP addresses is strictly reserved for loopback testing and cannot be assigned to a standard physical network interface?

  • ◯ A) 127.0.0.1
  • ◯ B) 10.0.0.1
  • ◯ C) 169.254.1.1
  • ◯ D) 224.0.0.1
Show Answer & Rationale

Correct Answer: A) 127.0.0.1

Rationale: The entire /8 block starting with 127 is reserved for this internal testing purpose and never routes traffic to the physical network.

Hint: This specific first-octet value is referenced heavily in operating system network diagnostics to refer to the local machine itself.

4. Which IP address falls within the RFC 1918 private range equivalent to a Class B network scale?

  • ◯ A) 172.16.5.10
  • ◯ B) 10.15.0.5
  • ◯ C) 192.168.1.1
  • ◯ D) 169.254.0.5
Show Answer & Rationale

Correct Answer: A) 172.16.5.10

Rationale: The 172.16.0.0/12 block is designated as the private range for mid-sized networks, scaling up to roughly one million addresses.

Hint: Recall the three main RFC 1918 blocks and associate them with their historical class letters A, B, and C based on their size.

5. A routing protocol sends an update to the 224.0.0.5 address. What type of traffic is this?

  • ◯ A) Multicast
  • ◯ B) Unicast
  • ◯ C) Broadcast
  • ◯ D) Anycast
Show Answer & Rationale

Correct Answer: A) Multicast

Rationale: The 224.0.0.0/4 range, historically known as Class D, is designed specifically for one-to-many communication to subscribed receivers.

Hint: Consider the purpose of the 224.0.0.0/4 block, which allows a single sender to efficiently reach a specific subscribed group rather than everyone.

6. During the DHCP DORA process, which message does the client broadcast to let all listening servers know which specific IP lease it has chosen?

  • ◯ A) Request
  • ◯ B) Discover
  • ◯ C) Offer
  • ◯ D) Acknowledge
Show Answer & Rationale

Correct Answer: A) Request

Rationale: The client broadcasts its selection so the chosen server can commit the lease and unselected servers can return their offered IPs to the pool.

Hint: At this stage, the client has already received potential IP addresses from servers and must officially claim one.

7. How does Port Address Translation (PAT) allow multiple private IP addresses to share a single public IP address?

  • ◯ A) By assigning unique ephemeral port numbers to each private session
  • ◯ B) By translating MAC addresses directly to public IP addresses
  • ◯ C) By encapsulating the private IPv4 packet inside an IPv6 header
  • ◯ D) By dynamically rotating through a large pool of public IP addresses
Show Answer & Rationale

Correct Answer: A) By assigning unique ephemeral port numbers to each private session

Rationale: The router rewrites the source IP and source port, keeping track of the return path using the port number to route traffic back to the correct internal host.

Hint: Think about the 'Hotel Ritz switchboard' analogy and how an extension number helps route a call.

8. Applying the RFC 5952 shortening rules, how should the IPv6 address 2001:0db8:0000:0000:0000:0000:0000:0001 be represented?

  • ◯ A) 2001:db8::1
  • ◯ B) 2001:0db8::1
  • ◯ C) 2001:db8::0001
  • ◯ D) 2001:db8::0::1
Show Answer & Rationale

Correct Answer: A) 2001:db8::1

Rationale: Leading zeros are dropped from the groups, and the continuous block of zeros in the middle is replaced by a single double colon.

Hint: Apply both the rule for omitting leading zeros in a group and the rule for replacing the longest sequence of all-zero groups.

9. At what point during a DHCP lease does the client typically attempt a 'Short DORA' renewal via unicast to the original server?

  • ◯ A) At the T1 timer, representing 50% of the lease duration
  • ◯ B) At the T2 timer, representing 87.5% of the lease duration
  • ◯ C) Immediately after the lease fully expires
  • ◯ D) When the client is rebooted manually
Show Answer & Rationale

Correct Answer: A) At the T1 timer, representing 50% of the lease duration

Rationale: The client attempts a targeted renewal directly with the server that originally granted the address to extend its time without causing broadcast noise.

Hint: This occurs relatively early in the lease lifecycle to ensure connectivity is maintained without waiting for an emergency state.

10. What is the primary purpose of Supernetting (route summarization)?

  • ◯ A) To merge contiguous smaller network blocks into a single summary route
  • ◯ B) To divide a large IP block into smaller segments for security boundaries
  • ◯ C) To allow multiple private IP addresses to share a single public IP address
  • ◯ D) To dynamically assign IP configurations to host devices on the network
Show Answer & Rationale

Correct Answer: A) To merge contiguous smaller network blocks into a single summary route

Rationale: This aggregation reduces the overall size of routing tables, resulting in faster lookups and fewer routing updates across the network.

Hint: Think about how large organizations simplify the routing information they advertise to the rest of the world to increase efficiency.
