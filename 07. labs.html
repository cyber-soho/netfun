Alternative Open-Source Discovery Tools

Masscan

Crazy-fast TCP scanner; can sweep the entire internet in 6 minutes (do not).

RustScan

Modern Rust wrapper over Nmap; finds open ports in seconds, feeds to Nmap.

Angry IP

Friendly GUI; great for absolute beginners doing host discovery only.

netdiscover

Pure ARP-based discovery — great for finding hosts on switches that block ICMP.

arp-scan

Same idea, smaller scope; gold standard for "what is on my LAN right now".

Alternative Open-Source Firewall Tools

Beyond Cisco IOS ACLs, these platforms are industry standards for routing and security.

pfSense / OPNsense
Full FreeBSD-based firewall distros with web UIs. OPNsense offers faster release cycles and stricter defaults. Replaces the Cisco router for SoHo.
VyOS
An open-source router OS with Cisco-style CLI — perfect if you liked the ACL syntax today.
OpenWrt
Open-source router firmware that runs on real consumer hardware. Great for hardening home networks.
Suricata / Zeek
Network analysis and IDS/IPS frameworks. They sit next to the firewall to watch for malicious patterns and generate rich logs.
